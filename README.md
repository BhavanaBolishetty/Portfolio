# Bhavana Bolishetty – Portfolio

A modern, responsive personal portfolio website built to showcase my background, skills, and projects as a **Software Engineer / Full-Stack Developer**.

This portfolio highlights my professional experience, technical skill set, and selected projects, with a clean UI and smooth user interactions.

---

## Features

- Responsive design (desktop, tablet, mobile)
- Active navigation highlighting based on scroll position
- Dark / Light theme toggle
- Animated typing effect for roles
- Smooth scroll and reveal animations
- Projects displayed in uniform cards
- Contact form with email delivery (EmailJS)
- Clean, accessible UI with modern CSS styling

---

## Tech Stack

**Frontend**
- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- JavaScript (Vanilla JS)

**Libraries / Services**
- EmailJS (Contact form email delivery)
- Font Awesome (Icons)

**Tools**
- Git & GitHub
- GitHub Pages (Deployment)
